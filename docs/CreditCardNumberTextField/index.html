Class Credit​Card​Number​Text​Field 

@objc(SPRCreditCardNumberTextField) open class CreditCardNumberTextField: SecureTextField

Handles credit card number input from the user.

Only accepts numbers. When input matches a known card brand pattern, input will be limited to the max allowable characters for that brand and shows the brand icon at the trailing edge. Formats the input into number groups while the user types. Unless editing, the field's content will be masked with the maskCharacter showing only the last four digits.
